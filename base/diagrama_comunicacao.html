# Introdução

Diagrama de comunicação, também chamado de diagrama de colaboração tem como propósito mostrar as interações UML entre objetos e/ou partes usando mensagens sequenciadas de forma livre. Este diagrama corresponde a um diagrama de sequenciai simples e sem mecanismos de estruturação. Analisamos nosso diagrama de classes e constituímos o diagrama a partir dos processos tentando trazer mais clareza para as cardialidades. Utilizamos a ferramenta lucidchar.
